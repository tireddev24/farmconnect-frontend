# FarmConnect
A platform that connects farmers and buyers

## Tools Used
You need to have all these installed for everything to work fine
- Node.js - JavaScript run-time environment
- VSCode - Integrated Development Environment

### How To Use The Repo
1. Install all required tools
2. Clone the repo by running this in your VS Code terminal or command line

``` bash
git clone https://github.com/tireddev24/farmconnect-frontend.git
```

3. To work on the frontend
   
   Install dependencies  
    ```bash
     npm install
    ```
   Run the frontend server 
    ```bash
   npm run dev  
    ```
The site should be available in your browser @ `http://localhost:5173`
